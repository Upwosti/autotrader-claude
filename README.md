# AutoTrader Claude

Automated MT5 Trading Bot powered by Claude AI.

## Setup
- Python 3.11
- MetaTrader5, pandas, numpy, flask, scikit-learn
- Telegram notifications via python-telegram-bot
- GitHub integration via PyGithub

## Structure
- main.py - Entry point
- 	rader/ - Trading logic
- strategies/ - Trading strategies
- utils/ - Utilities
